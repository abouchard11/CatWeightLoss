## How It Works

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  QR Code    │     │  Branded    │     │   Weight    │     │  One-Tap    │
│  on Your    │ ──► │  App Opens  │ ──► │  Tracking   │ ──► │  Reorder    │
│  Packaging  │     │  (Your Logo)│     │  + Portions │     │  (Your Link)│
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

Customer scans → Your branded app → Tracks their cat's weight → Reorders YOUR food

## The App Experience

**Dashboard**
- Cat profile with photo
- Weight progress ring
- Trend graph (last 7 entries)
- Quick actions: Log Weight, Log Activity, Feeding Plan

**Portion Calculator**
- Daily calorie needs (RER × activity factor)
- Portion size per meal
- Configured for your food's kcal/gram

**Reorder Flow**
- "Running low?" prompt
- One tap to Chewy/Petco/your site
- Affiliate tracking for attribution
