# EXHIBIT B: ENGAGEMENT METRICS SPECIFICATION

## Metrics Collected

All metrics are aggregated and anonymized. No personally identifiable information is collected or stored.

| Metric | Definition | Granularity |
|--------|------------|-------------|
| Activations | Count of QR Code deep link launches | Per SKU, Daily |
| Completions | Count of Quick Setup flow completions | Per SKU, Daily |
| App Opens | Count of app launches post-setup | Per SKU, Daily |
| Weight Entries | Count of weight log entries | Per SKU, Daily |
| 3+ Entry Users | Count of users with 3+ weight entries | Per SKU, Weekly |
| Progress Bucket | Distribution of users by progress % (0/25/50/75/100) | Per SKU, Weekly |

## Derived Metrics (Calculated)

| Metric | Formula |
|--------|---------|
| Completion Rate | Completions / Activations |
| 3+ Entry Rate | 3+ Entry Users / Completions |
| Avg Days Between Logs | Sum(days between entries) / Count(entries) |

## Data Not Collected

The following data is intentionally **NOT** collected:

| Excluded Data | Reason |
|---------------|--------|
| Actual weight values | Medical liability |
| Individual user journeys | Privacy |
| Feeding compliance | Blame risk |
| Cat names or photos | PII |
| Location data | Privacy |
| Device identifiers (raw) | Privacy — only hashed |

## Data Retention

- Engagement Metrics: 24 months, then automatically purged (extended retention available upon request)
- Device hash: Non-reversible, changes on app reinstall
- No data transmitted off-device at any time

## Reporting

Licensor shall provide Licensee with:

- Weekly metrics summary via email
- Monthly executive dashboard (PDF)
- Raw data export upon request (CSV or JSON format)
